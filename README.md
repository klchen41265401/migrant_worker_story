# 移工經歷網站（最簡易版）

## 專案概述

E 與移工認識經過的真實紀實網站，記錄 2024 年 1 月至 2026 年 3 月間的故事。

## 檔案架構

```
移工經歷網站(最簡易)/
├── index.html    # 主網站頁面（時間軸敘事，RWD 響應式設計）
├── 經歷.md       # 原稿（完整手稿記錄）
└── README.md     # 本檔案
```

## 技術細節

- **純靜態 HTML**：單一 `index.html` 檔案，無需伺服器
- **CSS 內嵌**：Google Fonts (Noto Sans TC)、CSS 變數主題色、時間軸排版
- **RWD 響應式**：支援手機與桌面瀏覽
- **功能**：頂部進度條、黏性導覽列、回到頂部按鈕、滾動追蹤高亮

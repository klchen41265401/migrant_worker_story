## 技術細節

- **架構**：靜態 HTML + JavaScript 動態載入（區段化 SPA）
- **樣式**：獨立 CSS 檔案（`css/style.css`），使用 CSS 變數管理主題色彩
- **字型**：Google Fonts — Noto Sans TC
- **互動**：原生 JavaScript 實作捲動導覽、進度條、月份高亮、內容動態載入
- **響應式**：支援桌面與行動裝置瀏覽
- **部署**：GitHub Pages（直接從 `main` 分支部署，無需建構工具）

## 內容說明

網站以月份為單位，逐月呈現事件卡片，涵蓋：

- 日常生活事件與里程碑
- 法律程序（筆錄、傳票）
- 財務管理與契約紀錄
- 人際關係互動與情感變化
- 工作轉換與搬遷紀錄

## 使用方式

### GitHub Pages（推薦）

直接推送至 GitHub，啟用 Pages 即可瀏覽。`index.html` 會自動透過 `fetch()` 載入 `sections/` 下的內容片段。

### 本機預覽

因為使用 `fetch()` 載入 HTML 片段，本機預覽需要 HTTP 伺服器（不能直接用 `file://` 開啟）：

```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js npx
npx serve .

# 使用 VS Code Live Server 擴充套件
# 直接右鍵 index.html → Open with Live Server
```

然後在瀏覽器開啟 `http://localhost:8000`。

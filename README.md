# 移工經歷網站

一頁式時間軸網站，記錄小恩與移工（戎狄）從 2024 年 1 月至 2026 年 4 月的相處經歷。

## 專案結構

```
移工經歷網站/
├── index.html                        # 主網頁（單頁式時間軸，含內嵌 CSS/JS）
├── 人物側寫.md                        # 戎狄生命中重要人物的背景資料
├── 人物側寫1.md                       # 補充人物側寫（戎狄爸、姊、三峽前女友、蘆洲女）
├── 20260404與移工現任女友的對話.md      # 小恩與戎狄現任女友的警告對話紀錄
├── 20260405狀態新增.md                # 2026-04-05 行為紀錄與狀態更新
├── 戎狄心理狀態分析_20260405.md        # 戎狄心理狀態綜合分析（中立醫療觀點）
├── 戎狄心理狀態分析_毒辣版_20260405.md  # 戎狄心理狀態分析（超級毒辣版）
├── 戎狄心理狀態分析_終極毒辣真實版_20260405.md # 戎狄心理分析（終極毒辣真實版，Gemini 產出）
├── README.md                         # 本說明文件
```

## 技術細節

- **架構**：純靜態 HTML 單頁應用，無需後端
- **樣式**：內嵌 CSS，使用 CSS 變數管理主題色彩
- **字型**：Google Fonts — Noto Sans TC
- **互動**：原生 JavaScript 實作捲動導覽、進度條、月份高亮
- **響應式**：支援桌面與行動裝置瀏覽

## 內容說明

網站以月份為單位，逐月呈現事件卡片，涵蓋：

- 日常生活事件與里程碑
- 法律程序（筆錄、傳票）
- 財務管理與契約紀錄
- 人際關係互動與情感變化
- 工作轉換與搬遷紀錄

## 使用方式

直接以瀏覽器開啟 `index.html` 即可瀏覽。
